El Metodo de Aprendizaje

El Metodo de Aprendizaje de Inteligencia Artificial en FreeCodingTour se basa en los siguientes principios:

  1. Entender la programacion como:
    Entrada -> Proceso -> Salida
    donde el Proceso puede ser visto como una caja negra
  2. Programacion Guiada por Casos de Prueba
  3. Entender el Machine Learning como Programacion Automatica guiada por Casos de Prueba
  4. Entender el Deep Learning como:
    Entrada en Lenguaje Humano -> Proceso -> Salida en Formato Deseado
    donde Lenguaje Humano puede ser Texto, Imagenes o Audio
