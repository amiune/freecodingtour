Free Coding Tour

Aqui encontraras cursos de programacion faciles, gratuitos y sin publicidad para aprender desde cero sin conocimientos previos y sin instalar ningun software ni necesitar una computadora especial

  • Para que sea gratuito y sin publicidad he utilizado github y github pages.
  • Para que sea interactivo y no se necesite comprar una pc especial ni instalar software especial he utilizado google colab.
  • Para que sea accesible para alguien que no tiene conocimientos previos he dejado la parte matematica cargo de python y utilizado este como una calculadora potente. Es decir, la multiplicacion de matrices, el calculo de derivadas o el calculo del minimo de una funcion son operaciones que python resolvera por mi.
